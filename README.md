# Yemeni-Doors
موقع تعريفي أنيق وسريع التحميل، يركز على إبراز العلامة التجارية ويعكس فخامة المنتج من خلال البساطة والوضوح في التصميم.

YemeniDoors مشروع قمت بتصميمه وتطويره لعرض خدمات شركة متخصصة في صناعة وتركيب الأبواب. يتميز الموقع بتصميم عصري بسيط يعتمد على الألوان الفاخرة (الأسود والذهبي)، مع استخدام تأثيرات الحركة (Scroll Animations) لزيادة التفاعل البصري. يتضمن الموقع أقساماً رئيسية مثل: الصفحة الرئيسية، من نحن، الخدمات، معرض الصور، ونموذج تواصل متكامل.
